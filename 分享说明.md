## 🚀 分享给别人的步骤

### 1. 发送打包文件

将 `feishu-wechat-auto-publisher-share.tar.gz` 发送给对方

### 2. 告知安装步骤

```bash
# 解压
tar -xzf feishu-wechat-auto-publisher-share.tar.gz

# 进入目录
cd feishu-wechat-auto-publisher

# 复制配置模板
cp config.json.template config.json

# 编辑配置（填入各自的 API 密钥）
vim config.json

# 安装依赖
pip3 install -r scripts/requirements.txt

# 测试运行
python3 scripts/feishu-wechat-auto.py <飞书文档链接>
```

- ✅ 微信公众号（订阅号/服务号）的 `app_id` 和 `app_secret`
- ✅ 飞书开放平台应用的 `app_id` 和 `app_secret`
- ✅ 飞书群机器人 webhook URL

## 📋 配置指南（给对方）

### 微信公众号配置

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 开发 → 基本配置
3. 获取 `AppID` 和 `AppSecret`

### 飞书应用配置

1. 登录 [飞书开放平台](https://open.feishu.cn/)
2. 应用开发 → 企业自建 → 创建应用
3. 凭证与基础信息 → 获取 `App ID` 和 `App Secret`
4. 权限管理 → 添加权限：
   - `docx:document`（读取文档）
   - `wiki:wiki`（读取知识库）
   - `drive:file`（访问云文档）

### 飞书 webhook 配置

1. 在飞书群 → 群设置 → 机器人 → 添加机器人
2. 选择"自定义机器人"
3. 复制 webhook URL
